## 🚀 What is Face AI Analyzer?

> **One selfie → Personality, Age, Aura, Symmetry, Celebrity Match, Past Life & more.**
> No login required. No photo uploads stored. Instant AI results. Free forever.

**Face AI Analyzer** is a Progressive Web App (PWA) that uses AI to analyze faces directly in the browser — no server upload, no privacy risk, completely free.

🔗 **Try it live:** [aifaceanalyzer.netlify.app](https://aifaceanalyzer.netlify.app)

## ⚡ Quick Start

```bash
# 1️⃣ Clone the repository
git clone https://github.com/yourusername/face-ai-analyzer.git
cd face-ai-analyzer

# 2️⃣ Install dependencies
npm install

# 3️⃣ Setup environment variables
cp docs/.env.example .env
# → Fill in your API keys in .env file

# 4️⃣ Run locally
node js/server.js

# 5️⃣ Open in browser
# → http://localhost:3000
```

## 🔑 Environment Variables

Copy `docs/.env.example` → create `.env` and fill in:

```env
# Payment Gateway (Razorpay)
RAZORPAY_KEY_ID=YOUR_RAZORPAY_LIVE_KEY
RAZORPAY_KEY_SECRET=YOUR_RAZORPAY_SECRET

# Firebase (Pro user tracking)
FIREBASE_API_KEY=YOUR_FIREBASE_API_KEY
FIREBASE_PROJECT_ID=YOUR_FIREBASE_PROJECT_ID
FIREBASE_APP_ID=YOUR_FIREBASE_APP_ID

# Analytics
GA_MEASUREMENT_ID=YOUR_GA_MEASUREMENT_ID

# AdSense
ADSENSE_PUB_ID=YOUR_ADSENSE_PUB_ID
AD_SLOT_1=YOUR_AD_SLOT_1
AD_SLOT_2=YOUR_AD_SLOT_2
AD_SLOT_3=YOUR_AD_SLOT_3
```

> ⚠️ **Never commit your `.env` to GitHub — it's in `.gitignore` already ✅**

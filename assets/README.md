# 🖼️ Assets — Icons & Images

This folder contains all visual assets used in the Face AI app.

---

## 📂 File List

| File | Size | Purpose |
|---|---|---|
| `favicon.ico` | 254 KB | Browser tab icon, Google search icon, bookmarks |
| `icon-192.png` | 192×192 px | Android home screen icon, PWA install icon |
| `icon-512.png` | 512×512 px | High-res PWA icon, splash screen |

---

## 📱 Where Each Icon Is Used

### `favicon.ico`
- Browser tab
- Google search results
- Browser bookmarks & history
- Windows taskbar (when pinned)

### `icon-192.png`
- Android home screen (after PWA install)
- Apple iPhone/iPad home screen
- PWA install prompt thumbnail

### `icon-512.png`
- PWA splash screen (loading screen)
- High-resolution displays
- App stores (if listed)
